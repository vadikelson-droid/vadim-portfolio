Laravel + Vue.js

План обучения для Full-stack разработчика

ФАЗА 1

Основы PHP

1-2 недели

Синтаксис и основы

  • Переменные, типы данных, операторы
  • Условия, циклы, функции
  • Массивы и строки (array_map, array_filter, implode/explode)
  • ООП: классы, наследование, интерфейсы, трейты
  • Namespaces и autoloading (PSR-4)
  • Composer — менеджер пакетов PHP
php.net docs + Laracasts "PHP for Beginners"

У тебя уже есть

  • Python/JS опыт — PHP синтаксис учится быстро
  • ООП — есть из Python (классы в Telethon, aiohttp)
  • npm → Composer (аналогичная концепция)
ФАЗА 2

Laravel — Основы

2-3 недели

Архитектура Laravel

  • Установка: composer create-project laravel/laravel
  • Структура проекта (routes, controllers, models, views)
  • Artisan CLI — генерация кода (make:model, make:controller)
  • Маршрутизация (web.php, api.php)
  • Контроллеры и Request/Response
  • Blade-шаблонизатор (views)
Laravel Docs + Laracasts "Laravel from Scratch"

Eloquent ORM + Миграции

  • Миграции — создание/изменение таблиц (как схемы в MongoDB)
  • Модели — аналог Mongoose в Laravel
  • Отношения: hasMany, belongsTo, belongsToMany
  • Seeder и Factory — заполнение тестовыми данными
  • Query Builder и Eloquent запросы
laravel.com/docs/eloquent

MySQL

  • Установка MySQL / использование SQLite для разработки
  • Основы SQL: SELECT, INSERT, UPDATE, DELETE, JOIN
  • Индексы и связи (foreign keys)
  • Миграции Laravel заменяют ручной SQL
MySQL Tutorial + Laravel Migrations

Практика

Создать простой CRUD-блог: статьи, категории, комментарии. Маршруты, контроллеры, Eloquent, миграции.

ФАЗА 3

Laravel — REST API

2 недели

API Development

  • API маршруты (routes/api.php)
  • Resource Controllers (apiResource)
  • Form Request Validation
  • API Resources — форматирование JSON-ответов
  • Sanctum — аутентификация по токенам
  • Middleware (auth, cors, throttle)
Laravel API Tutorial + Sanctum Docs

У тебя уже есть

  • REST API опыт из webhook серверов (aiohttp)
  • JSON форматирование из receipt_formatter.py
  • Auth концепции из Firebase Auth (Nova Rent)

Практика

Переписать Dental Bot API на Laravel: CRUD для пациентов, врачей, записей. Авторизация через Sanctum. Тестирование через Postman.

ФАЗА 4

Vue.js 3 — Основы

2-3 недели

Основы Vue 3

  • Создание проекта: npm create vue@latest (Vite)
  • Компоненты (.vue файлы: template + script + style)
  • Реактивность: ref(), reactive(), computed()
  • Директивы: v-if, v-for, v-model, v-bind, v-on
  • Props и Emit — общение между компонентами
  • Lifecycle hooks: onMounted, onUpdated
vuejs.org + Vue Mastery "Intro to Vue 3"

Composition API

  • setup() и <script setup> (рекомендуемый стиль)
  • Composables — переиспользуемая логика
  • watch() и watchEffect()

У тебя уже есть

  • Продвинутый JS — Vue учится намного быстрее
  • Компонентный подход — был в GSAP/Three.js проектах
  • Реактивность — понимание из Firebase Firestore

Практика

Создать Todo-приложение на Vue 3: добавление, удаление, фильтрация, localStorage. Потом — список карточек (как в Nova Rent) на Vue.

ФАЗА 5

Vue.js — Продвинутый уровень

2 недели

Экосистема Vue

  • Vue Router — маршрутизация SPA (как в Nova Rent, но встроенная)
  • Pinia — state management (замена Vuex)
  • Axios — HTTP-клиент для запросов к Laravel API
  • Vite — сборщик проекта
Pinia Docs + Vue Router Docs

Интеграция с Laravel

  • Vue как отдельный SPA + Laravel как API
  • CORS настройка
  • Axios interceptors для токена авторизации
  • Загрузка файлов через API
  • Пагинация (Laravel paginate + Vue компонент)

Практика

Подключить Vue frontend к Laravel API из Фазы 3. Полный CRUD с авторизацией. Роутинг, Pinia store, Axios.

ФАЗА 6

Финальный проект

2-3 недели

Full-stack приложение (Laravel + Vue)

  • Система управления рестораном (близко к твоему опыту)
  • Авторизация (login/register) — Laravel Sanctum
  • Меню с категориями — CRUD + фильтрация
  • Онлайн-заказы — корзина, оформление
  • Админ-панель — управление заказами и меню
  • Уведомления — Laravel Events + Broadcasting
  • Деплой на VPS (Nginx + PHP-FPM + MySQL)

Результат

Готовый full-stack проект в портфолио. Можно показать работодателю как доказательство навыков Laravel + Vue.
